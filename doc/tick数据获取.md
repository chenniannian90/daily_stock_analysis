# A股 Tick 数据（逐笔成交）获取

## 概述

Tick 数据指股票的逐笔成交明细，包含每笔交易的成交时间、价格、成交量、成交金额及买卖方向。

本系统通过 akshare 的腾讯财经接口获取 Tick 数据。

---

## 接口说明

### `ak.stock_zh_a_tick_tx_js(symbol)`

| 参数 | 说明 |
|------|------|
| `symbol` | 股票代码，格式 `sz000001`（深交所）或 `sh600519`（上交所） |

| 返回字段 | 说明 |
|----------|------|
| `成交时间` | 格式 `HH:MM:SS`，从 09:25 集合竞价到 15:00 收盘 |
| `成交价格` | 该笔成交单价（元） |
| `价格变动` | 相对上一笔的价格变动（元） |
| `成交量` | 该笔成交量（股） |
| `成交金额` | 该笔成交金额（元） |
| `性质` | 买盘 / 卖盘 / 中性盘 |

### 调用示例

```python
import akshare as ak

df = ak.stock_zh_a_tick_tx_js(symbol='sz000001')  # 平安银行
print(df.head())
```

输出：

```
       成交时间   成交价格  价格变动    成交量      成交金额  性质
0  09:25:00  11.25  0.00   8564   9634028  卖盘
1  09:30:00  11.25  0.00  12859  14476361  卖盘

3  09:30:06  11.22 -0.01   3283   3683883  卖盘
4  09:30:09  11.21 -0.01   7965   8931272  卖盘
```

---

## 限制

- **仅支持当日数据**，无法获取历史 Tick。接口只接受股票代码参数，不含日期参数。
- **单次单只股票**，不支持批量查询。
- 腾讯接口约 8~10 秒返回一次，批量获取需控制频率避免被封。

## 获取历史 Tick 的替代方案

如需历史 Tick 数据，可考虑以下付费数据源：

| 数据源 | 说明 |
|--------|------|
| tushare pro | 高级会员可获取历史逐笔成交，需积分 |
| Wind 万得 | 机构级数据终端，覆盖全量历史 Tick |
| JoinQuant 聚宽 | 线上回测平台，部分 Tick 数据可用 |
| RiceQuant 米筐 | 同上，回测环境自带 Tick 数据 |
