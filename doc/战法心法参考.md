## 一、龙头战法核心体系

### 1.1 筹码峰战法（四大形态）

| 形态 | 特征 | 操作 |
|------|------|------|
| 低位单峰密集 | 价格低位、筹码集中、单峰结构 | 放量突破 = 买入 |
| 底部锁仓 + 多峰延续 | 底部筹码不松动，上方新峰形成 | 坚定持有，回踩即买 |
| 高位单峰 + 筹码松动 | 高位筹码集中但开始散开 | 卖出信号 |
| 多峰套牢 + 低位无新峰 | 上方套牢盘重重，下方无新资金建仓 | 观望 |

口诀：**上峰不移，下跌不止；下峰锁定，行情未止。单峰突破，趋势加速；多峰延续，回踩即买。**

### 1.2 龙头切换心法

不执念于单一龙头，永远站在资金流动的一侧。新热点三大信号：

1. 早盘多股强势涨停
2. 对应板块整体涨幅 ≥ 2%
3. 板块涨停股数量连续三天递增

### 1.3 龙头分歧低吸（通达信指标逻辑）

同时满足四个条件触发信号：
- 突破 30 天箱体
- 站上强势上轨
- 放量 ≥ 1.8 倍
- 7 天内首次启动

适用场景：龙头二波、趋势主升、箱体突破启动。

### 1.4 龙头股生命周期

```
建仓 → 洗盘 → 启动 → 主升 → 见顶 → 下跌
```

核心思路：识别当前所处阶段，只在启动和主升阶段参与。

## 二、新生代交易方法论

### 2.1 选股优先级链

```
找龙头 → 做不了龙头做补涨 → 做不了补涨做低吸
```

### 2.2 四个唯一性（资金聚焦标准）

| 唯一性类型 | 说明 |
|------------|------|
| 容量唯一性 | 大资金唯一能进出的标的 |
| 高标唯一性 | 市场上唯一的高度板 |
| 梯队唯一性 | 板块内部唯一完整涨停梯队 |
| 逻辑唯一性 | 题材中最正宗的标的 |

### 2.3 交易节奏

- **买入分歧**：分歧转一致时介入（有资金主动承接）
- **卖出高潮**：高潮时离场（主力喜欢在此下车）

### 2.4 风控铁律

| 规则 | 说明 |
|------|------|
| 本金安全第一 | 跌 50% 需涨 100% 才回本，必须设止损 |
| 杜绝追涨杀跌 | 宁可错过，不可做错 |
| 不加杠杆 | 新手不借钱炒股 |
| 仓位控制 | 最多持有 5 只股票，反复操作 |
| 不参与监管票 | 严重异动的票尽量远离 |

### 2.5 学习路径

1. 研究龙虎榜 → 拆解各游资手法 → 内化为自己的模式
2. 每天写交易日志（买入逻辑 + 卖出理由）
3. 先用 1/10 本金练手，攒够 100 笔经验再逐步加仓
4. 深耕一种方法，不要在低吸/打板/价投之间来回跳

## 三、炒股养家心法（游资传奇）

### 3.1 六大总纲

| 编号 | 心法 | 解读 |
|------|------|------|
| 1 | 基于市场情绪揣摩 | 判断风险与收益的比较，以此指导操作 |
| 2 | 高手买入龙头，超级高手卖出龙头 | 卖点把握比买点境界更高 |
| 3 | 别人贪婪我更贪婪，别人恐慌我更恐慌 | 顺势而为，不与趋势对抗 |
| 4 | 敢于大盘低位空仓，敢于大盘高位满仓 | 心中无顶底，操作自随心 |
| 5 | 永不止损，永不止盈 | 只有进场与出局，买入机会卖出不确定性 |
| 6 | 得散户心者得天下 | 人气所向，牛股所在 |

### 3.2 市场情绪与博弈

**本质**：股票走势 = 持币者与持筹者的博弈循环。

- 赚钱效应 → 吸引场外资金 → 追涨者连续赚钱 → 自信增强加仓 → 热点持续
- 亏钱效应 → 引发恐慌抛售 → 追涨者连续失手 → 减少操作 → 高位股补跌

**阶段适配**：
- 强势阶段 → 买强（资金不断进入，强标被选中概率大）
- 弱势阶段 → 低吸（等恐慌盘宣泄完毕，少量买盘即可推动上涨）

### 3.3 仓位 = 赢面的函数

| 赢面 | 仓位 |
|------|------|
| < 60% | 观望 |
| 60%-70% | 小仓出击 |
| 70%-80% | 中仓出击 |
| 80%-90% | 大仓出击 |
| ≥ 90% | 满仓 |

满仓前置条件：上涨空间 ≥ 30%-50%，下跌空间 ≤ 3%-5%，且有"后面还有3到5个涨停"的判断。

### 3.4 十条交易铁律

**防守六条：**

| 铁律 | 说明 |
|------|------|
| 判错即离 | 走势证伪逻辑 → 立即止损，不抱幻想 |
| 永远分仓 | 不孤注一掷，投资 ≠ 赌博 |
| 不接下落的刀 | 右侧信号确认前绝不抄底 |
| 大赚后空仓 | 大盈后多巴胺过度，强制冷却 |
| 只在行情健康时入场 | 没有赚钱效应，"不交易"就是最好的交易 |
| 严守系统边界 | 只做认知范围内的信号 |

**进攻四条：**

| 铁律 | 说明 |
|------|------|
| 吃鱼身 | 放弃鱼头（萌芽期）和鱼尾（多头陷阱），只取主升浪 |
| 追逐新热点 | A股喜新厌旧，新题材 = 更少套牢盘 + 更大想象空间 |
| 量价齐升是唯一标准 | 缩量上涨是耍流氓，无量上涨极易崩盘 |
| 分批建仓，一次性清仓 | 买入可分段，卖出必须果断 |

### 3.5 养家经典语录

> 掌握市场之心，胜利接踵而至；心被市场掌握，失败连绵不绝。

> 买入机会，卖出风险，只做对的交易，胜负交给概率。多一份淡定，少一分胜负心。

> 忘记你的成本。假设你现在空仓，你会怎么办——如果还会买持仓股就拿着，会买别的就换股，会选择空仓就卖出。

> 大局观是核心——站在更高高度看市场，一旦领悟，受益无穷。

### 3.6 养家的极简盘面

不看复杂指标，只看两样东西：
- **K 线**
- **成交量**

其余指标基本不看。重点是揣摩"哪些在涨、哪些在跌、涨跌之后怎么走"。

## 五、真龙头四维判定体系

> 以下四条是量化龙头股的核心标准，由外而内逐层筛选。

### 5.1 判定链：指数 → 板块 → 个股 → 盘口

```
指数方向  ──→  板块（强于指数的才是主线板块）
板块主线  ──→  个股（强于板块其他个股的才是龙头）
龙头个股  ──→  连板高度（有高度的才有辨识度）
高度龙头  ──→  分歧转一致（禁得起分歧考验的才是真龙头）
```

### 5.2 四条核心标准

| 层级 | 标准 | 量化指标 | 判定方式 |
|------|------|----------|----------|
| **1. 带动效应** | 能带动板块甚至指数的才是真龙头 | 带动率 = 个股涨幅/板块涨幅 > 2 | 龙头涨停时，板块跟涨；龙头炸板，板块跳水 |
| **2. 强中选强** | 强于指数的板块 × 强于板块的个股 | 板块涨幅 > 指数涨幅，个股涨幅 > 板块涨幅 | 三层筛选：指数基准 → 板块超额 → 个股超额 |
| **3. 连板高度** | 龙头有连板高度，辨识度靠高度积累 | 连板数 ≥ 2，且是市场最高标或板块最高标 | 高度打开空间，吸引接力资金 |
| **4. 分歧转一致** | 禁得起分歧，分歧后转一致 | 分歧日（高换手+宽振幅）→ 次日缩量加速 | 分歧=买盘和卖盘充分博弈后多头胜出 |

### 5.3 三层筛选实现（代码对应）

`src/services/dragon_strategy_service.py` 中的 `identify_dragon_stocks()` 实现了前三层：

```
Level 1: 板块强于指数?    sector_pct > benchmark_index_pct
Level 2: 个股强于板块?    stock_pct  > sector_pct
Level 3: 个股涨停?        stock_pct  >= 9.9%

真龙头 = 三层全过 (levels_passed == 3)
```

**第四层（连板高度 + 分歧转一致）依赖涨停板历史数据（东财接口被封，待网络恢复后补充）。**

### 5.4 真龙头评分公式

| 维度 | 权重 | 计算方式 |
|------|------|----------|
| 板块地位 | 0-25分 | 25 - (板块排名-1) × 4 |
| 个股超额收益 | 0-30分 | min(30, (个股涨幅-板块涨幅) × 3 + 5) |
| 板块超额收益 | 0-20分 | min(20, (板块涨幅-指数涨幅) × 4) |
| 涨停加成 | 0-25分 | 涨停=25, 涨幅≥7%=15, 涨幅≥5%=8 |

- 分歧转一致分 (0-15分): 分歧日换手>15%+次日缩量涨停 = +15
